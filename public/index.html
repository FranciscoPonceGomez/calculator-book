How to build a 2050 calculator

This book explains how to write your own 2050 calculator.

It is work in progress. Please send feedback to 2050pathways@decc.gsi.gov.uk

It is available in four formats:

  1. Online
  2. PDF
  3. epub — for use in ebook readers
  4. Word
